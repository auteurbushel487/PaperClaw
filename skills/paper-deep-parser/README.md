# paper-deep-parser

Deep parsing Skill for the paper-reading-agent pipeline.

## Overview

Extracts structured JSON fields from knowledge card Markdown files (card.md) generated by `read-arxiv-paper`. Part of the Agent-orchestrated paper pipeline (Phase 4).

## Architecture

- **Agent orchestration**: Agent triggers `read-arxiv-paper` per paper via `sessions_spawn`, then calls `card_parser.py` to extract structured fields
- **Deterministic tool**: `card_parser.py` handles Markdown parsing and field extraction
- **Graceful degradation**: Unknown fields filled with `"N/A"`

## Scripts

| Script | Purpose |
|--------|---------|
| `$PAPER_AGENT_ROOT/scripts/card_parser.py` | Parse card.md into structured JSON |

## Input/Output

- **Input**: `pipeline_data/{run_id}/skill3_final_selection.json` (papers to parse)
- **Output**: `pipeline_data/{run_id}/skill4_parsed/{arxiv_id}.json` (per-paper structured data)

## Extracted Fields

- `sub_field`: Research sub-field classification
- `ID_paradigm`: Item ID/representation paradigm
- `item_tokenizer`: Tokenization method
- `baselines_compared`: Baseline methods compared
- `transferable_techniques`: Techniques transferable to other work
- `inspiration_ideas`: Research ideas inspired by the paper
